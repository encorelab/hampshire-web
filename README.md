# The Stack
* middleman
* haml
* sass

#Setup
* `git pull https://github.com/encorelab/hampshire-web.git`
* `cd hampshire-web`
* to start the middleman server type`middleman`

# Building/Compiling a project
* within `hampshire-web` folder type `middleman build` to build project
